redis知识汇总[初级]--数据类型（String）

易语

框架 / redis

redis知识汇总[初级]–数据类型（String）

1.append

（1）命令：APPEND key value

（2）介绍：

① key存在，value将会追加到原有值的后面
② key不存在，将初始化设置该值（输入的设置）

127.0.0.1:6379> append myname yiyu
(integer) 4
127.0.0.1:6379> get myname
"yiyu"
127.0.0.1:6379> append myname isman
(integer) 9
127.0.0.1:6379> get myname
"yiyuisman"

2.bitcount

（1）命令：BITCOUNT key [start] [end]

（2）介绍：

① key存在，返回被设置为 1 的位的数量
② key不存在，当做字符串处理，故为0
③ -1 表示最后一个位，而 -2 表示倒数第二个位

//不存在
127.0.0.1:6379> bitcount bcname
(integer) 0
//存在
127.0.0.1:6379> setbit sbname 20 1
(integer) 0
127.0.0.1:6379> bitcount sbname
(integer) 1

3.bitop

（1）命令：BITOP operation destkey key [key …]

（2）介绍：

① key存在，对一个或多个保存二进制位的字符串 key 进行位元操作，并将结果保存到 destkey 上
② key不存在，默认字符串为0
③ operation：AND（一个或多个逻辑并）、OR（一个或多个逻辑或）、NOT（此只能单个key，逻辑非）、XOR（一个或多个逻辑异或）

//key不存在
127.0.0.1:6379> BITOP AND and-result bit
(integer) 0
//key存在
127.0.0.1:6379> setbit topbit1 1 1
(integer) 0
127.0.0.1:6379> setbit topbit2 3 1
(integer) 0
127.0.0.1:6379> BITOP AND and-result topbit1 topbit2
(integer) 1

4.decr

（1）命令：DECR key

（2）介绍：

① key存在，改对应value减1
② key不存在，value初始化0后减1
③ value为不能加减的数，报错

//key不存在
127.0.0.1:6379> decr decrcount2
(integer) -1
//key存在
127.0.0.1:6379> set decrcount1 100
OK
127.0.0.1:6379> decr decrcount1
(integer) 99

5.decrby

（1）命令：DECRBY key decrement

（2）介绍：

① key存在，改对应value减decrement
② key不存在，value初始化0后减decrement
③ value为不能加减的数，报错

//key不存在
127.0.0.1:6379> decrby drcount 100
(integer) -100
//key存在
127.0.0.1:6379> set drcount1 50
OK
127.0.0.1:6379> decrby drcount1 2
(integer) 48

6.get

（1）命令：GET key

（2）介绍：

① key存在，返回对应value的值
② key不存在，返回nil
③ key非字符串，报错

//key不存在
127.0.0.1:6379> get name1
(nil)
//key存在
127.0.0.1:6379> set gname yiyu
OK
127.0.0.1:6379> get gname
"yiyu"

7.getbit

（1）命令：GETBIT key offset

（2）介绍：

① key存在，获取偏移量对应的value
② key不存在，返回0（offset>value 也会返回0）

// key不存在返回0
127.0.0.1:6379> getbit gbit 20190606
(integer) 0
//对存在的offset 获取
127.0.0.1:6379> setbit sbit 20190606 1
(integer) 1
127.0.0.1:6379> getbit sbit 20190606
(integer) 1

8.getrance

（1）命令：GETRANGE key start end

（2）介绍：

key存在，返回该范围的字符串
key不存在，返回””
-1 表示最后一个字符， -2 表示倒数第二个

//已存在key
127.0.0.1:6379> set rangename yiyu
OK
127.0.0.1:6379> get rangename
"yiyu"
//第一个字符为0开始
127.0.0.1:6379> getrange rangename 1 2
"iy"
//key不存在返回“”
127.0.0.1:6379> getrange rangename1 1 2
""

9.getset

（1）命令：GETSET key value

（2）介绍：

key存在，设置新值，返回旧值（非字符串报错）
key不存在，返回nil

127.0.0.1:6379> getset gsname yiyu
(nil)
127.0.0.1:6379> getset gsname yiyuhahaha
"yiyu"

11.incr

（1）命令：INCR key

（2）介绍:

key存在：值增加1
key不存在：初始化值0，在执行incr

//对已存在key加一
127.0.0.1:6379> set incrname1 20
OK
127.0.0.1:6379> get incrname1
"20"
127.0.0.1:6379> incr incrname1
(integer) 21
//对不存在key初始化0加一
127.0.0.1:6379> incr incrname2
(integer) 1

12.incrby

（1）命令：INCRBY key increment

（2）介绍:

key存在：值增加increment
key不存在：初始化值0，在执行incr

//对已存在的值加5
127.0.0.1:6379> set incrbyname 10
OK
127.0.0.1:6379> get incrbyname
"10"
127.0.0.1:6379> incrby incrbyname 5
(integer) 15
//对不存在的值初始化0加5
127.0.0.1:6379> incrby incrbyname1 5
(integer) 5

13.incrbyfloat

（1）命令：INCRBYFLOAT key increment

（2）介绍:

与incrby同理，支持float数据

14.mget

（1）命令：MGET key [key …]

（2）介绍:

key存在：返回value
key不存在：不存在的key返回nil

127.0.0.1:6379>  msetnx mname1 1 mname2 2
(integer) 0
127.0.0.1:6379> mget mname1 mname2 mname2
1) "1"
2) "2"
3) (nil)

15.msetnx

（1）命令：MSETNX key value [key value …]

（2）介绍:

key存在：即使存在一个也会全部失败
key不存在：设置多个key-value
此操作为原子性操作

//key未存在
127.0.0.1:6379> msetnx mname1 1 mname2 2 mname3 3
(integer) 1
127.0.0.1:6379> mget mname1 mname2 mname3
1) “1”
2) “2”
3) “3”
//存在个别key（存不进会失败但不会影响已存在的值）
127.0.0.1:6379> msetnx mname4 4 mname1 1
(integer) 0
127.0.0.1:6379> mget mname4 mname1
1) (nil)
2) “1”

16.psetex

(1) 命令：PSETEX key milliseconds value

（2）介绍：

与setex类似，只是时间换成毫秒

17.set

（1）命令：SET key value [EX seconds] [PX milliseconds] [NX|XX]

（2）介绍:

key存在：value覆盖原值
key不存在：关联key与value
EX seconds：key过期时间，单位秒
PX millisecond：key过期时间，单位毫秒
NX：等同命令SETNX key value（可以不存在时起作用）
XX：与上同理

//设值
127.0.0.1:6379> set myname yiyu
OK
127.0.0.1:6379> get myname
"yiyu"
//覆盖原值
127.0.0.1:6379> set myname goodman
OK
127.0.0.1:6379> get myname
"goodman"
//设值时间
127.0.0.1:6379> set myname yiyu EX 20
OK
127.0.0.1:6379> GET myname
"yiyu"
//20s后
127.0.0.1:6379> get myname
(nil)

18.setbit

（1）命令：SETBIT key offset value

（2）介绍：

① key存在，设值偏移量的值
② key不存在，生成新的字符串
③ offset：偏移量，大于或等于0，小于2^32

//可用来记录当日是否登陆
127.0.0.1:6379> setbit sbit 20190606 1
(integer) 0
127.0.0.1:6379> getbit sbit 20190606
(integer) 1

19.setex

（1）命令：SETEX key seconds value

（2）介绍：

原子性操作
key存在，value覆盖旧值
key不存在，设置该值
seconds ：设值key过期时间，单位秒

127.0.0.1:6379> setex exname 20 yiyu
OK
127.0.0.1:6379> get exname
"yiyu"
//已过期
127.0.0.1:6379> get exname
(nil)

20.setnx

（1）命令：SETNX key value

（2）介绍：

key存在，不做任何操作
key不存在，设置该值

//设值（key不存在）
127.0.0.1:6379> setnx nxname yiyu
(integer) 1
127.0.0.1:6379> get nxname
"yiyu"
//设置值（key存在）
127.0.0.1:6379> setnx nxname hahaha
(integer) 0
127.0.0.1:6379> get nxname
"yiyu"

21.setrance

（1）命令：SETRANGE key offset value

（2）介绍：

key存在，从偏移量开始的位置设值
key不存在，当做空白字符串处理设值

//对已存在的key设值
127.0.0.1:6379> set rname yiyuhahaha
OK
127.0.0.1:6379> setrange rname 5 lalala
(integer) 11
127.0.0.1:6379> get rname
"yiyuhlalala"
//对未存在的key设值
127.0.0.1:6379> setrange empty_name 2 "yiyu"
(integer) 6
//  该偏移前的位置被空格替代
127.0.0.1:6379> get empty_name
"\x00\x00yiyu"

22.strlen

（1）命令：strlen key

（2）介绍:

key存在：返回对应value的长度（value不是字符串返回错误）
key不存在：返回0

127.0.0.1:6379> strlen myname
(integer) 9
127.0.0.1:6379> strlen myname1
(integer) 0
127.0.0.1:6379>
